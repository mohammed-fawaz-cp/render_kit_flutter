# renderkit

Custom DSL and Flutter preview components for RenderKit.

## Features

This package contains the core DSL widgets and classes for RenderKit, including:
- Actions and state management models.
- Layout and styling properties.
- Dynamic widgets that get parsed and compiled to Kotlin Compose and Swift SwiftUI.
- Preview widgets to render the DSL locally in Flutter.

## Usage

Add `renderkit` as a dependency in your `pubspec.yaml`:

```yaml
dependencies:
  renderkit: ^0.1.0
```

Import the package and define widgets using the RenderKit DSL:

```dart
import 'package:renderkit/renderkit.dart';

// Your declarative UI widgets...
```
