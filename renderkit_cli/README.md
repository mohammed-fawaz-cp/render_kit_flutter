# renderkit_cli

CLI tool for RenderKit to validate environments, configure pipelines, and compile widgets.

## Features

This command line tool allows you to:
- Diagnose your development environment for compilation requirements.
- Set up target configurations.
- Trigger pipeline operations.

## Usage

You can run `renderkit_cli` via the Dart SDK:

```bash
dart run renderkit_cli <command> [arguments]
```
